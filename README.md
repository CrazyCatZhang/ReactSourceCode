# React Source Code

## Native debugging

### Install dependencies

```shell
npm install
```

### Running debugging environments for various React versions

Support `v18`, `v17`, `17.0.0-alpha.0`, `v16.13.1`, `v16.12.0`

#### Commands

```shell
npm run dev:<version>
```
